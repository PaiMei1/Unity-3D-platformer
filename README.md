# 3D Platformer - Unity

## Description
This is a 3D platformer / obby-style game developed in Unity using C#. The game features obstacle-based gameplay where players navigate platforms, jumps, and challenges to reach the end.

## Features
- 3D player movement and jumping mechanics  
- Obby-style obstacle course levels  
- Physics-based interactions and collisions  
- Level design with platform challenges  
- Checkpoint or progression systems
- Camera and control system  
- Additional gameplay features and mechanics  

## Technologies Used
- Unity Engine  
- C#  
- Visual Studio  

## Purpose
This project was created to practice:
- 3D game development in Unity  
- Physics-based movement and interaction  
- Level design and obstacle creation  
- C# scripting and gameplay systems
